# Screenshot Naming

Bu klasore gercek uygulama ekranlarini asagidaki adlarla at.
Desteklenen uzantilar: `.png`, `.jpg`, `.jpeg`, `.webp`, `.avif`

Gerekli dosya adlari:

- `dashboard`
- `lessons`
- `calendar`
- `ai`
- `reports`

Ornek:

- `dashboard.png`
- `lessons.png`
- `calendar.png`
- `ai.png`
- `reports.png`

Sayfa acildiginda otomatik olarak bu dosyalari bulur.
Dosya yoksa varsayilan mockup gorselleri kullanilir.
